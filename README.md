# 🌊 InkTide: Digital Book Archive

**InkTide** is a high-performance web application designed for searching, tracking, and accessing digital literature. The platform integrates metadata from the **Open Library API** with the direct-access capabilities of the **Internet Archive**.

---

### 🎯 Project Purpose
The objective of this project is to provide a unified interface for the discovery of books and the immediate retrieval of public-domain digital assets. It functions as both a cataloging tool and a gateway to free reading materials, bridging the gap between book metadata and actual reading access.

### ✨ Planned Features
* **Optimized Search:** Efficient book discovery using debounced input handling.
* **Reading Management:** Tools to categorize books as *Want to Read*, *Currently Reading*, or *Read*.
* **Instant Access:** Automated identification of available EPUB/PDF downloads for public-domain works.
* **Data Persistence:** Storage of user preferences and library data using the **Web Storage API**.
* **Responsive UI:** A mobile-first design featuring a system-aware **Dark Mode**.

### 🛠️ Technical Stack

| Component | Technology |
| :--- | :--- |
| **Frontend** | HTML5 / CSS3 (Flexbox & Grid) |
| **Logic** | Vanilla JavaScript (ES6+) |
| **Data Source** | Open Library REST API |
| **Constraint** | 100% Zero-Loop Policy (Array HOFs only) |

### 🚀 Setup & Execution
1. Clone the repository to the local machine.
2. Execute `index.html` in a standard web browser.

*Note: No external dependencies or build tools are required for execution.*
